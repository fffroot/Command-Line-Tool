> 💻 Коллекция консольных утилит и инструментов для автоматизации на разных языках.  
> От простых скриптов и парсеров до мощных TUI-приложений и CLI-фреймворков.

## 📖 О репозитории
`Command-Line-Tool` — это единое пространство для утилит, которые упрощают рутину, ускоряют разработку или решают специфичные задачи в терминале.  
Каждый подпроект **автономен**: имеет собственные зависимости, систему сборки и документацию. Объединяет их только формат — запуск из командной строки.
